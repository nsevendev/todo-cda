# TodoCda  

> Api pour l'application todo-cda pour les cours de la formation cda  

## Prérequis  

- `.env` => copier/coller le fichier `.env.example` et le renommer en `.env`, renseigner les variables d'environnement
- `api/appsettings.Developpement.json` => demander le fichier à un membre de l'équipe  

## Installation  

- voir toute les commandes make tapez `make` dans le terminal  

```bash
# start container mode dev
$ make dev

# arret et destruction des containers
$ make down
```
